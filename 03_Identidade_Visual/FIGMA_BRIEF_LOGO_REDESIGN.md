  Instrucao Passo a Passo no Figma                                                                                                                                             
                                                                                                                                                                               
  Frame 1: Criar a Estrutura Base                                                                                                                                              
  1. Novo arquivo: "DeepWork Logo Redesign"                                                                                                                                    
  2. Frame 800x800px, fundo branco                                                                                                                                             
  3. Ativar Layout Grid 8px para precisao                                                                                                                                      
                                                                                                                                                                               
  Frame 2: Desenhar o Icone                                                                                                                                                    
  1. Pen Tool (P)                                                                                                                                                              
  2. Desenhar tres hastes verticais:                                                                                                                                           
    - Largura de cada haste: 48px                                                                                                                                              
    - Altura: 280px                                                                                                                                                            
    - Espacamento entre hastes: 24px                                                                                                                                           
    - Corner radius nas pontas: 24px                                                                                                                                           
  3. Conectar na base:                                                                                                                                                         
    - Angulo de 45 graus                                                                                                                                                       
    - Formar um "V" invertido sutil                                                                                                                                            
    - Base mais larga que o topo (estabilidade = warehouse)                                                                                                                    
  4. Adicionar diferenciacao nas camadas:                                                                                                                                      
    - Haste esquerda: curva no topo para dentro (entrada de dados)                                                                                                             
    - Haste central: reta com topo arredondado (processamento)                                                                                                                 
    - Haste direita: curva no topo para fora (saida/output)                                                                                                                    
                                                                                                                                                                               
  Frame 3: Aplicar Gradiente                                                                                                                                                   
  1. Selecionar todas as formas                                                                                                                                                
  2. Fill > Linear Gradient                                                                                                                                                    
  3. Ponto superior: #0A3D2E                                                                                                                                                   
  4. Ponto medio (50%): #1A5C45                                                                                                                                                
  5. Ponto inferior: #2D8B6F                                                                                                                                                   
  6. Angulo: 180 graus                                                                                                                                                         
                                                                                                                                                                               
  Frame 4: Criar Variantes                                                                                                                                                     
  1. Duplicar icone (Ctrl+D)                                                                                                                                                   
  2. Criar versoes:                                                                                                                                                            
    - Gradient (principal)                                                                                                                                                     
    - Solid Dark: #0A3D2E                                                                                                                                                      
    - Solid Light: #2D8B6F                                                                                                                                                     
    - White: #FFFFFF                                                                                                                                                           
  3. Converter cada um em Component (Ctrl+Alt+K)                                                                                                                               
  4. Combinar em Component Set                                                                                                                                                 
                                                                                                                                                                               
  Frame 5: Compor Logo Completo                                                                                                                                                
  1. Icone a esquerda                                                                                                                                                          
  2. Texto "DEEPWORK" - Montserrat Bold - #0A2342                                                                                                                              
  3. Texto "AI FLOWS" - Montserrat Medium - #1A5C45 (verde novo)                                                                                                               
  4. Espacamento entre icone e texto: 32px                                                                                                                                     
  5. Alinhar verticalmente ao centro
