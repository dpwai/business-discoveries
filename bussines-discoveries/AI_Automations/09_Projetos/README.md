## 📝 Template de Projeto

Para criar um novo projeto, use a seguinte estrutura:

```
XX_Nome_Cliente/
├── README.md                    # Visão geral do projeto
├── 00_SUMARIO_REUNIAO.md        # Sumário da primeira reunião
├── 01_DIAGNOSTICO_AS_IS_TO_BE.md # Diagnóstico As-Is/To-Be
├── 02_INFORMACOES_CLIENTE.md    # Informações do cliente
├── 03_ACOMPANHAMENTO.md         # Acompanhamento do projeto
├── 04_PROPOSTA_COMERCIAL.md     # Proposta comercial (futuro)
├── 05_REQUISITOS_TECNICOS.md    # Requisitos técnicos (futuro)
└── dados/                        # Pasta para dados do cliente (se necessário)
```

## 🎯 Tipos de Projeto

### Produto-Piloto
Projetos iniciais para validação de conceito e desenvolvimento de casos de uso.

### Projeto Comercial
Projetos com contrato formal e escopo definido.

### Projeto Interno
Projetos para desenvolvimento interno ou melhoria de processos.
