# Prompt de Mapeamento de Contexto & Estrutura de Landing Page

**Objetivo:**
Você atuará como um **Estrategista de Produto e Copywriter Sênior**. Sua missão é ler todo o contexto do projeto "Concessionária Kugler Veículos" e estruturar uma **Landing Page (LP) de Alta Conversão** que será usada como apresentação para o cliente (Daniel Kugler).

**1. Mapeamento de Contexto (Input):**
Leia e analise profundamente os seguintes arquivos para entender a dor, a solução e o mercado:
1.  `../README.md` (Visão Geral)
2.  `../01_DIAGNOSTICO_AS_IS_TO_BE.md` (Dores atuais vs. Sonho futuro)
3.  `../06_PESQUISA_MERCADO/RELATORIO_PESQUISA.md` (Dados de mercado, benchmarks, tendências)
4.  `../07_DESENVOLVIMENTO_CRIATIVO/BRAINSTORMING.md` (A solução técnica: Agente NBS, Tablet, etc.)
5.  `../04_PREPARACAO_PROXIMA_REUNIAO.md` (Estratégia de venda)

**2. Estrutura da Landing Page (Output Desejado):**
Com base no contexto absorvido, crie o conteúdo textual (Copy) para uma Landing Page seguindo esta estrutura de vendas:

*   **Hero Section (A Promessa):**
    *   Headline: Uma frase impactante que resume o benefício principal (ex: "Sua oficina rodando sozinha...").
    *   Subheadline: Explicação breve de como (Agente de Automação NBS).
    *   CTA (Call to Action): O próximo passo.

*   **O Problema (A Dor):**
    *   Descreva o cenário atual de caos, falta de dados e ineficiência, usando a linguagem do cliente.
    *   Use dados de mercado (do relatório de pesquisa) para mostrar que ele está perdendo dinheiro (ex: Cycle Time).

*   **A Solução (O Agente NBS):**
    *   Apresente o "Agente" não como um software, mas como um membro da equipe.
    *   Destaque as 3 funcionalidades chave: Check-in Inteligente, Controle de Mão de Obra Gamificado, Torre de Controle.

*   **Como Funciona (O Processo):**
    *   Passo a passo simples (1, 2, 3) de como o consultor e o técnico usam no dia a dia.

*   **Prova Social / Autoridade (Benchmarks):**
    *   Mostre que essa é a tendência mundial (Workshop 4.0).

*   **Benefícios (ROI):**
    *   O que ele ganha? (Tempo, Dinheiro, Paz de espírito).

*   **Fechamento (Próximos Passos):**
    *   Convite para o projeto piloto.

**Tom de Voz:**
*   Profissional, inovador, mas "pé no chão" (linguagem de dono de negócio).
*   Foque em **lucro e eficiência**, não apenas em tecnologia.
