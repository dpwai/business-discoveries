## 🆓 Opção 1: Starter (Free)

### O que inclui
- 3 arquivos Figma (limite total)
- 3 páginas por arquivo
- 30 dias de histórico de versão
- Viewers ilimitados
- Comentários e compartilhamento básico

### Funciona para DPWAI?

**❌ NÃO RECOMENDADO**

**Por quê:**
- 3 arquivos = você precisaria colocar todos os 9 dashboards em 1 arquivo
- Sem Dev Mode (João não consegue inspecionar código)
- Histórico de 30 dias = você perde versões antigas
- Parece "amador" para apresentar a clientes

**Custo:** R$ 0

---

## 💼 Opção 2: Professional

### O que inclui
- **Arquivos ilimitados**
- **Páginas ilimitadas por arquivo**
- **Histórico de versão ilimitado**
- **Libraries compartilhadas** (Design System)
- **Audio conversations**
- **Viewers ilimitados** (grátis)
- Dev Mode: +$25/mês/usuário (ou incluso para editores)

### Estrutura de Preço

| Tipo | Mensal | Anual (20% off) |
|------|--------|-----------------|
| Editor (Full) | $20/mês | $15/mês |
| Dev Mode Only | $25/mês | ~$21/mês |

### Cenário DPWAI

```
1 Editor (Rodrigo):     $15/mês (anual)
1 Dev Mode (João):      $25/mês
Viewers (Claudio, etc): $0 (grátis)
─────────────────────────────────────
TOTAL:                  $40/mês = ~R$ 240/mês
```

### Funciona para DPWAI?

**✅ RECOMENDADO PARA COMEÇAR**

**Prós:**
- Arquivos e páginas ilimitados = perfeito para 9 dashboards
- Libraries = você cria o Design System SOAL uma vez, reutiliza
- João acessa Dev Mode para implementar
- Stakeholders visualizam grátis
- Custo baixo (~R$ 240/mês)
- **Billing mensal disponível** = pode cancelar a qualquer momento

**Contras:**
- Se você contratar mais 1 designer, sobe para $55/mês
- Sem SSO (login via Google/Microsoft corporativo)
- Sem admin controls avançados

**Custo Anual:** ~R$ 2.880/ano

## 🏢 Opção 3: Organization

### O que inclui
- **Tudo do Professional, mais:**
- **Design System Analytics** (vê quais componentes são usados)
- **Branching e Merging** (tipo Git para design)
- **SSO (Single Sign-On)** com Google/Microsoft/Okta
- **Org-wide libraries** (centraliza tudo)
- **Advanced permissions** (controle granular)
- **Shared fonts** (fontes custom para toda org)
- **Audit logs** (histórico de ações)

### Estrutura de Preço

| Tipo | Preço |
|------|-------|
| Editor (Full) | $45/mês (apenas anual) |
| Dev Mode Only | $25/mês |
| Viewer | $0 (grátis) |

**⚠️ IMPORTANTE:** Não há billing mensal. Você paga 12 meses upfront.

### Cenário DPWAI

```
1 Editor (Rodrigo):     $45/mês × 12 = $540/ano
1 Dev Mode (João):      $25/mês × 12 = $300/ano
Viewers:                $0
─────────────────────────────────────
TOTAL ANUAL:            $840/ano = ~R$ 5.040/ano
```

### Funciona para DPWAI?

**⚠️ TALVEZ - Depende do seu crescimento**

**Prós:**
- Branching = trabalhar em variações sem quebrar o original
- Design System Analytics = ver se o Design System está sendo usado corretamente
- SSO = profissional para clientes enterprise

**Contras:**
- **3× mais caro** que Professional para o mesmo cenário
- Features de enterprise que você provavelmente não usa agora
- Commitment anual (sem mensal)
- Overengineering para 1-2 editores

**Quando faz sentido:**
- Quando DPWAI tiver 5+ designers
- Quando clientes exigirem SSO
- Quando Design System for crítico (múltiplos projetos reutilizando)

**Custo Anual:** ~R$ 5.040/ano

## 💡 Dicas de Economia

### 1. Billing Anual (20% off)
Se você tem certeza que vai usar por 1 ano, pague anual:
- Professional: $20/mês → $15/mês (economia de $60/ano por editor)

### 2. Dev Mode vs Editor Full
João precisa só de Dev Mode ($25/mês), não de Editor Full ($15-20/mês). Parece mais caro, mas:
- Dev Mode = acesso a specs, CSS, assets
- Editor Full = criar/editar designs (João não precisa)

**Nota:** Se João também for criar protótipos, considere Editor Full ($15) que inclui Dev Mode.

### 3. Viewers são Grátis
Claudio, Tiago, Valentina não precisam pagar nada. Eles:
- Visualizam os designs
- Comentam diretamente nos frames
- Aprovam via comentários

### 4. Educacional (Se Aplicável)
Se você ou alguém da equipe é estudante/professor, Figma Professional é **GRÁTIS**.

### 5. Starter para Testes
Você pode começar com Starter (grátis) para testar o workflow antes de pagar. Limitação: 3 arquivos.

## ❓ FAQ

**Q: Posso começar com Free e migrar depois?**
A: Sim! Seus arquivos migram automaticamente. Só não perca tempo em limitações do Free.

**Q: Se eu contratar um designer freelancer temporário, como funciona?**
A: Você adiciona como Editor ($15-20/mês), trabalha no projeto, remove quando terminar. Billing é proporcional.

**Q: João precisa de Dev Mode ou Editor?**
A: Se João só vai inspecionar specs e exportar assets → Dev Mode ($25/mês).
Se João também vai criar/editar protótipos → Editor Full ($15/mês com Dev Mode incluso).

**Q: E se eu usar Superset ao invés de código custom?**
A: Você ainda precisa do Figma para criar os designs visuais. Superset não tem designer visual como Figma.

**Q: FigJam está incluso?**
A: Sim! FigJam (whiteboard) está incluso em todos os planos pagos.
