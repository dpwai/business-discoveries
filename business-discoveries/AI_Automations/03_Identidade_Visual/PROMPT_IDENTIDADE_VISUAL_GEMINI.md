# Prompt para Identidade Visual - Gemini 3.0

## 🎯 OBJETIVO DO PROMPT

Criar um prompt completo e estruturado para o Gemini 3.0 desenvolver a identidade visual completa da marca **Deepwork AI Flows**, incluindo logo, paleta de cores, tipografia e diretrizes de uso.

---

## 📋 PROMPT COMPLETO PARA GEMINI 3.0

### PARTE 1: CONTEXTO DA MARCA

```
Você é um designer de identidade visual especializado em marcas B2B de tecnologia e consultoria.

Preciso criar a identidade visual completa para a marca "Deepwork AI Flows", uma empresa de automação consultiva que desenvolve soluções específicas para processos de médias e grandes empresas.

CONTEXTO DA EMPRESA:
- Nome: Deepwork AI Flows
- Tagline: "Soluções personalizadas para problemas específicos"
- Posicionamento: Empresa de automação consultiva (híbrido: consultoria + tecnologia)
- Público-alvo: Médias e grandes empresas brasileiras (50-500+ funcionários)
- Setores: Construção, Consórcios, Indústria, Concessionárias

MISSÃO:
Aumentar a capacidade produtiva de médias e grandes empresas através de automações inteligentes desenvolvidas especificamente para seus processos únicos.

VISÃO:
Ser referência em automação consultiva no Brasil, reconhecida por criar soluções específicas que transformam a capacidade produtiva de médias e grandes empresas.

VALORES:
1. Resultado Mensurável - Foco em valor tangível
2. Consultoria Profunda - Entender antes de criar
3. Especificidade - Cada solução é única
4. Parceria e Colaboração - Trabalhar junto
5. Integração Natural - Usar o que já existe
6. Melhoria Contínua - Evoluir constantemente
7. Transparência - Comunicação clara e honesta

TOM DE VOZ:
- Profissional (mas não excessivamente formal)
- Consultivo (foco em entender e ajudar)
- Direto e claro (vai direto ao ponto)
- Focado em resultado (conecta a benefícios mensuráveis)
- Específico, não genérico (personalizado para cada cliente)

DIFERENCIAÇÃO:
- Consultoria profunda antes de desenvolver
- Soluções específicas, não genéricas
- Usa o que o cliente já tem (sem migração)
- Foco em resultado mensurável
- Melhoria contínua e supervisão
```

---

### PARTE 2: CONCEITOS VISUAIS A COMUNICAR

```
CONCEITOS QUE A IDENTIDADE VISUAL DEVE COMUNICAR:

1. PROFUNDIDADE / TRABALHO PROFUNDO
   - "Deepwork" = trabalho profundo, consultoria profunda
   - Não superficial, mas sim mergulho profundo
   - Visual: Camadas, profundidade, mergulho

2. FLUXOS / PROCESSOS
   - "Flows" = fluxos, processos, automações
   - Movimento, continuidade, fluxo
   - Visual: Linhas fluidas, setas, movimento

3. ESPECIFICIDADE / PERSONALIZAÇÃO
   - Cada solução é única
   - Não genérico, mas específico
   - Visual: Elementos únicos, não padronizados demais

4. INTEGRAÇÃO NATURAL
   - Conecta com o que já existe
   - Integração sem fricção
   - Visual: Conexões, integração, harmonia

5. INTELIGÊNCIA / IA
   - "AI" = inteligência artificial
   - Tecnologia moderna, mas acessível
   - Visual: Elementos que sugerem inteligência, rede, conexões

6. PROFISSIONALISMO B2B
   - Empresa séria e confiável
   - Adequada para médias e grandes empresas
   - Visual: Profissional, mas não corporativo demais

7. CONSULTORIA
   - Abordagem consultiva, não apenas técnica
   - Parceria, colaboração
   - Visual: Acolhedor, mas profissional
```

---

### PARTE 3: DIRETRIZES DE ESTILO

```
ESTILO VISUAL DESEJADO:

GERAL:
- Moderno, mas não excessivamente futurista
- Profissional B2B, mas acessível
- Limpo e organizado, mas não minimalista demais
- Sofisticado, mas não elitista
- Tecnológico, mas humano

NÃO QUEREMOS:
- ❌ Muito corporativo/tradicional
- ❌ Muito futurista/tech demais
- ❌ Muito minimalista (pode parecer genérico)
- ❌ Muito colorido/descontraído (não adequado B2B)
- ❌ Estilo "startup" muito jovem

QUEREMOS:
- ✅ Moderno e profissional
- ✅ Sofisticado mas acessível
- ✅ Tecnológico mas humano
- ✅ Específico, não genérico
- ✅ Profissional B2B adequado
```

---

### PARTE 4: ELEMENTOS DO LOGO

```
ELEMENTOS QUE PODEM SER INCLUÍDOS NO LOGO:

1. TIPOGRAFIA
   - Nome: "Deepwork AI Flows"
   - Pode ter variação: "Deepwork" + "AI Flows" separados
   - Tagline pode estar incluído ou não

2. ELEMENTOS VISUAIS SUGERIDOS (escolher 1-2):
   - Fluxos/linhas fluidas (representa "Flows")
   - Camadas/profundidade (representa "Deepwork")
   - Conexões/rede (representa integração e IA)
   - Seta/movimento (representa fluxo e progresso)
   - Forma geométrica única (representa especificidade)
   - Combinação de elementos acima

3. ESTILO DO LOGO:
   - Pode ser: Tipográfico, Pictórico, Combinado, Emblema
   - Deve funcionar: Horizontal, vertical, ícone isolado
   - Deve ser: Escalável, legível em pequeno e grande tamanho

4. VERSÕES NECESSÁRIAS:
   - Logo completo (com nome)
   - Ícone/Isotipo (apenas símbolo)
   - Versão horizontal
   - Versão vertical (se necessário)
   - Versão monocromática (preto/branco)
   - Versão para fundo escuro
   - Versão para fundo claro
```

---

### PARTE 5: PALETA DE CORES

```
DIRETRIZES PARA PALETA DE CORES:

SENTIMENTOS QUE AS CORES DEVEM PASSAR:
- Profissionalismo
- Confiança
- Inovação (mas não excessiva)
- Profundidade
- Integração/Harmonia
- Tecnologia (mas acessível)

SUGESTÕES DE CORES (baseado no contexto):

OPÇÃO 1: Azul Profundo (Profundidade + Confiança)
- Cor primária: Azul profundo (representa profundidade, confiança)
- Cor secundária: Azul claro (representa fluxo, movimento)
- Cor de destaque: Laranja/Amarelo (representa energia, resultado)

OPÇÃO 2: Azul + Verde (Profundidade + Crescimento)
- Cor primária: Azul profundo
- Cor secundária: Verde (representa crescimento, fluxo)
- Cor de destaque: Azul claro

OPÇÃO 3: Azul + Roxo (Profundidade + Inovação)
- Cor primária: Azul profundo
- Cor secundária: Roxo (representa inovação, IA)
- Cor de destaque: Azul claro

OPÇÃO 4: Tons de Azul (Monocromático Profissional)
- Variações de azul (profundo, médio, claro)
- Mais conservador, muito profissional
- Pode ser muito genérico

CORES A EVITAR:
- ❌ Vermelho (muito agressivo para B2B consultivo)
- ❌ Amarelo muito vibrante (pode parecer descontraído demais)
- ❌ Cores muito saturadas (não adequado B2B profissional)
- ❌ Paleta muito colorida (pode parecer genérico)

CORES NEUTRAS:
- Preto, cinza escuro, cinza médio, cinza claro, branco
- Para textos, fundos, elementos secundários
```

---

### PARTE 6: TIPOGRAFIA

```
DIRETRIZES PARA TIPOGRAFIA:

CARACTERÍSTICAS DESEJADAS:
- Moderna, mas não excessivamente futurista
- Profissional e legível
- Funciona bem em B2B
- Não muito decorativa
- Boa legibilidade em diferentes tamanhos

SUGESTÕES DE ESTILO:
- Sans-serif moderna (mais comum em tech)
- Pode ter serifa se for elegante e moderna
- Deve comunicar: Profissionalismo + Modernidade + Acessibilidade

FONTES SUGERIDAS (para referência):
- Inter, Poppins, Montserrat (modernas, acessíveis)
- Roboto, Open Sans (limpas, profissionais)
- Outras: A definir pelo designer

APLICAÇÃO:
- Logo: Fonte específica para marca
- Títulos: Fonte principal (bold/medium)
- Corpo de texto: Fonte secundária (regular/light)
- Pode usar mesma família com diferentes pesos
```

---

### PARTE 7: ELEMENTOS VISUAIS ADICIONAIS

```
ELEMENTOS VISUAIS PARA IDENTIDADE COMPLETA:

1. ÍCONES/ILUSTRAÇÕES:
   - Estilo: Moderno, limpo, profissional
   - Pode ter: Elementos de fluxo, conexões, processos
   - Evitar: Muito decorativo, muito infantil

2. PADRÕES/TEXTURAS:
   - Pode ter: Padrões sutis de fluxos/linhas
   - Evitar: Muito chamativo, distrai do conteúdo

3. FOTOS/IMAGENS:
   - Estilo: Profissional, real, não stock photos genéricos
   - Foco: Pessoas trabalhando, processos, resultados
   - Evitar: Fotos muito encenadas, genéricas

4. GRÁFICOS/DIAGRAMAS:
   - Estilo: Limpo, profissional, fácil de entender
   - Cores: Usar paleta da marca
   - Foco: Comunicar processos e fluxos
```
