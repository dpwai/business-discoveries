# Como Converter SVG para PNG

## 🎯 Solução Rápida: Usar Ferramenta Online

A forma mais rápida de converter os SVGs em PNGs é usar uma ferramenta online:

### Opção 1: CloudConvert (Recomendado)
1. Acesse: https://cloudconvert.com/svg-to-png
2. Faça upload do arquivo SVG
3. Configure:
   - **Width**: 400px (para logos horizontais) ou 200px (vertical) ou 120px (ícone)
   - **Height**: 120px (horizontal) ou 250px (vertical) ou 120px (ícone)
   - **Scale**: 2x (para alta resolução/retina)
4. Clique em "Convert"
5. Baixe o PNG

### Opção 2: Convertio
1. Acesse: https://convertio.co/svg-png/
2. Faça upload do arquivo SVG
3. Configure as dimensões
4. Converta e baixe

### Opção 3: SVG2PNG
1. Acesse: https://svgtopng.com/
2. Faça upload e converta

---

## 💻 Solução Local: Usar Inkscape (Windows)

### Instalação
1. Baixe Inkscape: https://inkscape.org/release/
2. Instale o programa

### Conversão via Linha de Comando
```batch
# Converter um arquivo
inkscape logo_conceito.svg --export-filename=logo_conceito.png --export-width=800 --export-height=240

# Ou usar o script batch fornecido
```

## 🖼️ Solução: Screenshot do HTML

1. Abra o arquivo `preview_logos.html` no navegador
2. Use a ferramenta de captura de tela do Windows (Win + Shift + S)
3. Capture cada logo individualmente
4. Salve como PNG
