# Estrutura de Pastas DPWAI

Padrão de organização para todos os repositórios da organização.

## Pastas

| Pasta | Descrição |
|-------|-----------|
| `00_CONHECIMENTO` | Base de conhecimento, documentação geral, wikis |
| `01_Visao_Negocio` | Visão estratégica, objetivos, metas, OKRs |
| `02_Identidade_Marca` | Branding, tom de voz, posicionamento |
| `03_Identidade_Visual` | Logos, cores, tipografia, assets visuais |
| `07_Operacional` | Processos, SOPs, checklists, workflows |
| `08_Ferramentas` | Scripts, tools, utilitários |
| `09_Projetos` | Documentação de projetos específicos |
| `10_Gestao_Tecnica` | Arquitetura, decisões técnicas, ADRs |
| `11_Juridico` | Contratos, termos, compliance |

## Arquivos Raiz

- `README.md` - Visão geral do repositório
- `ESTRUTURA_PASTAS.md` - Este arquivo (referência da estrutura)

## Notas

- Numeração permite ordenação visual consistente
- Gaps na numeração (04-06) são intencionais para futuras expansões
- Cada pasta deve ter seu próprio README.md quando apropriado
